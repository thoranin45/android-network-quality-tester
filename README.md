# 📡 SignalXpert

Android application for LTE/5G network quality testing.

## Features

- 🚀 Speed Test
- 🌐 Browsing Test
- 🎥 Video Streaming Test
- 📍 Base Station Mapping

## Tech Stack

- Kotlin
- Android SDK
- TelephonyManager
- ExoPlayer
- Volley
- OpenStreetMap
